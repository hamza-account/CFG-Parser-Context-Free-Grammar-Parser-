# CFG-Parser-Context-Free-Grammar-Parser-
A Context-Free Grammar (CFG) Parser developed using HTML, CSS, and JavaScript. This web-based application allows users to enter grammar rules and input strings to determine whether the strings are accepted or rejected according to the given CFG.

✨ Features
Enter custom Context-Free Grammar rules
Input strings for parsing
Validate strings based on the provided grammar
Display parsing results (Accepted/Rejected)
Simple and user-friendly interface
Responsive web design
Real-time input validation
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
🎯 Project Objective

The purpose of this project is to demonstrate the implementation of Context-Free Grammar parsing concepts while providing an interactive tool for learning and testing formal language and automata theory.

📂 How to Use
Clone this repository.
Open the project folder.
Launch index.html in your browser.
Enter the CFG production rules.
Type the input string.
Click the Parse button to check whether the string is accepted or rejected.
